# OpenQuant Trading Framework

Ein robustes, modulares Framework für quantitativen Handel und Backtesting.

## Projektstruktur

- **docs/**: Produktdokumentation (PRD, Roadmap, API-Referenz, Code-Standards).
- **tests/**: Vollständige Test-Suite (Unit-Tests, Integrationstests, E2E-Backtests).
- **config/**: Konfigurationsdateien für Tests und Code-Qualität.

## Installation

```bash
pip install -r requirements.txt
```

## Tests ausführen

```bash
pytest
```
